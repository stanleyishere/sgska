Wearable Sensor Casing

July 2021 - Present


This project is a collaborated work with mscLab, Faculty of Engineering UMS and Jarvis Software Technology, a start-up founded by my senior, Nyiak Tien Tang.

I worked on this project at the last semester of undergrad. This project is to conduct electromyography when the MyoWare muscle sensor is attached on user's hand or leg. The signals sent from the sensor(s) would be collected in a cloud storage system for further analysis. Since it is not practical to directly attach the sensor onto skin surface, I was responsible to design and develop a casing that can mount the sensor inside.
The ESP32-WROOM-32 soldered with MyoWare sensor.
I firstly design the casing via SolidWorks.

I made the casing via FDM technique. Printer used is Creality Ender 3 V-2. Below shows the printed casing with sensor.

As this project is still ongoing, more information will be documented.
